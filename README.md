# AStar
 
 A simple implementation of the popular Best-First Search Path Finding Algortihm, A*.
 Made with Unreal Engine 5 and C++.
 
### Features
 
 1. Multiple choices of distance functions - Euclidean, Manhattan, Octile and Chebyshev.
 2. Heap Optimization
